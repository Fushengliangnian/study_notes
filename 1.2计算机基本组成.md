- 指令和数据都是保存在存储器中:

  - 存储器基本组成(主存储器)
    - 存储体 - 存储单元组成 - 存储单元由存储原件组成 - 存储原件由 0/1 表示
      - 存储单元: 存放一串二进制代码
      - 存储字: 存储单元中二进制代码的组合
      - 存储字长: 存储单元中二进制代码的位长, 每个存储单元赋予一个地址
      - 存储单元是按地址寻访
    - MAR: 寄存器: 存储器地址寄存器: 反映存储单元的个数
    - MDR: 寄存器: 存储器数据寄存器: 反映存储字长

- 运算器的结构:

  - 运算器的功能

  - 运算器如何工作

    - 基本组成

      - ALU 算罗运算单元
      - ACC 寄存器
      - MQ 寄存器(乘商寄存机)
      - X

    - 操作过程

      - 加法操作的过程:

        - 指令:  + M

          初态: ACC 被加数

          [M] --> X

          [ACC] + [X] --> [ACC]

      - 减法操作: 

        - 指令:  - M

          初态 [ACC] 被减数

          [M] --> X

          [ACC] - [X]--> [ACC]

      - 乘法操作:

        - 指令: x M

          初态 ACC 被乘数

          [M] --> MQ

          [ACC] --> X

          0 --> ACC

          [x] x [MQ] --> ACC // MQ

      - 除法操作:

        - 指令: / M

          初态: ACC 被除数

          [M] --> X

          [ACC] / [x] --> MQ

          ​	余数在 ACC
